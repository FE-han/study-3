## 이름 길이 비교하기

### 문제 설명

- main.py의 코드를 확인하고 프로그램이 작동하도록 함수를 만들어보세요.

### 자료 구조

- **s1**
    - 타입 : 문자열
    - 저장 데이터 : 첫 번째 모자이름

- **s2**
    - 타입 : 문자열
    - 저장 데이터 : 두 번째 모자이름

### 풀이 과정

```txt
1. getlong 함수에 들어온 s1 ,s2 값의 길이를 서로 비교해서 길이가 더 긴 값을 반환

```

### 코드 구현
- 사용 언어 : 파이썬

```python
from function import getlong

s1 = input()
s2 = input()

print(getlong(s1, s2))


# getlong 함수를 만드세요.
def getlong(s1, s2):
    if len(s1) == len(s2):
        return
    elif len(s1) > len(s2):
        return s1
    else:
        return s2

```
