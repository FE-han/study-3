## 평균 구하기(평균 구하기!)

### 문제 설명

- 2개의 모자이름을 입력받아 두 모자중 이름의 길이가 긴 모자 이름 출력<br>



### 자료 구조

- s1, s2<br>
    - 타입 : str
    - 저장 데이터 : input()



### 풀이 과정
```txt
1. getlong 함수만들기

2. len함수를 이용해 비교하고 더 큰 숫자 리턴

```

### 코드 구현
```python
# 풀이과정 1,2
# getlong 함수를 만드세요.
def getlong(a,b):
    if len(a) < len(b):
        return b
    else :
        return a
```
