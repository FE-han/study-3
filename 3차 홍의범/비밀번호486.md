## 문제 설명

-  4, 6, 8로 이루어진 숫자들을 정해진 문자열로 출력 <br>


## 자료 구조
- nums  <br>
타입 : str<br>

- index  <br>
타입 : str<br>

## 풀이 과정
예시 :
```txt
1. 문자열을 입력받습니다.
2. 최종 값을 출력받을 문자열 index를 만든후
3. 반복문과 if elif 문을 이용하여 구현하였습니다.

```

## 코드 구현
```python
def yoonHa(value):

    index = ""
    for i in value:
        if i == "4":
            index += "love"

        elif i == "8":
            index += "smile"

        elif i == "6":
            index += "kiss"
    
    return index
        

nums = input()
print(yoonHa(nums))
```
