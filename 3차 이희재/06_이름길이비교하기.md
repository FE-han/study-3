
## 이름 길이 비교하기

### 문제 설명

- 모자 이름이 2개 들어오면, 모자 이름이 긴 모자를 출력<br>


### 자료 구조

- hat1<br>
    - 타입 : 문자열
    - 저장 데이터 : 모자1 이름 저장

- hat2<br>
    - 타입 : 문자열
    - 저장 데이터 : 모자2 이름 저장


### 풀이 과정

>-문제를 풀기 위한 풀이 과정을 작성합니다.<br>
>-풀이 과정을 단계별로 나누어 적거나, 의사 코드(pseudo code) 형태로 작성합니다.<Br>

```txt
1. 매개변수 hat2, hat2 가진 함수 getlong 생성

2. hat1 이름 길이가 hat2 보다 길면 hat1 리턴

3. 아니면 hat2 리턴

```
